# Adrenalin Rush for EXILED

![downloads](https://img.shields.io/github/downloads/Vretu-Dev/AdrenalinRush/total)

### Features:
- Set Movement Boost Speed
- Set Effect Duration

Minimum Exiled Version: 8.9.8

### Config:

```yaml
AR:
  is_enabled: true
  debug: false
  boost_duration: 3
  boost_intensity: 30
```
